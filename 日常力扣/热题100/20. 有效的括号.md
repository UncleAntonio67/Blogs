🏷简单 #栈 #字符串 

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

示例 1：

>输入：`s = "()"`
>输出：`true`

示例 2：

>输入：`s = "()[]{}"`
>输出：`true`

示例 3：

>输入：`s = "(]"`
>输出：`false`

示例 4：

>输入：`s = "([])"`
>输出：`true`

示例 5：

>输入：`s = "([)]"`
>输出：`false`

**提示：**

- `1 <= s.length <= 104`
- `s` 仅由括号 `'()[]{}'` 组成

# 解题思路与实现

- 利用栈匹配

| **维度**    | **分析内容**                                                                                                                                          |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| **核心思路**  | **栈的匹配逻辑**。<br>1. **遇到左括号**：说明后续需要一个匹配的右括号，将其压入 `stack`。<br>2. **遇到右括号**：它必须与当前栈顶的左括号匹配。如果不匹配或栈为空，则说明非法。<br>3. **最终检查**：遍历结束后，栈必须为空（确保所有左括号都被抵消）。 |
| **空间复杂度** | $O(n)$<br>在最坏情况下（如输入全是左括号），栈需要存储所有字符。                                                                                                             |
| **时间复杂度** | $O(n)$<br>只需对字符串进行一次线性扫描，每次栈操作（`push`/`pop`）都是 $O(1)$。                                                                                            |
| **优点**    | **逻辑严密**：通过 `mapping` 字典统一处理三类括号，代码扩展性强（比如增加 `<>` 也很容易）。                                                                                          |

## 方法一：

```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack = []
        mapping ={')':'(',']': '[', '}': '{'}
        for ch in s:
            if ch in mapping.values():
                stack.append(ch)
            else:
                if not stack or stack[-1] != mapping[ch]:
                    return False
                stack.pop()
        return not stack
```
