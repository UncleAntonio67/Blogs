给你一个字符串 `s`，找到 `s` 中最长的 回文 子串。

示例 1：

输入：s = "babad"
输出："bab"
解释："aba" 同样是符合题意的答案。
示例 2：

输入：s = "cbbd"
输出："bb"

**提示：**

- `1 <= s.length <= 1000`
- `s` 仅由数字和英文字母组成


# 解题思路与实现

对字符串里的每个字符进行“向两边扩展”，只要两边字符相同，就是回文。
你可以有两种中心：
1. 奇数回文中心：`i`（如 "aba"）
2. 偶数回文中心：`i, i+1`（如 "abba"）
## 方法一：

```python
class Solution:
    def longestPalindrome(self, s: str) -> str:
        if len(s) < 2:
            return s
        def expand(l,r):
            while l >= 0 and r < len(s) and s[l] == s[r]:
                l -= 1
                r += 1
            return s[l+1:r]
        res = ""
        for i in range(len(s)):
            res = max(res, expand(i,i), expand(i,i+1),key = len)
        return res
```
