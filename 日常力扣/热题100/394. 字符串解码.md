🏷中等 #栈 #递归 #字符串 

给定一个经过编码的字符串，返回它解码后的字符串。

编码规则为: `k[encoded_string]`，表示其中方括号内部的 `encoded_string` 正好重复 `k` 次。注意 `k` 保证为正整数。

你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式要求的。

此外，你可以认为原始数据不包含数字，所有的数字只表示重复的次数 `k` ，例如不会出现像 `3a` 或 `2[4]` 的输入。

测试用例保证输出的长度不会超过 `105`。

示例 1：

>输入：`s = "3[a]2[bc]"`
>输出：`"aaabcbc"`

示例 2：

>输入：`s = "3[a2[c]]"`
>输出：`"accaccacc"`

示例 3：

>输入：`s = "2[abc]3[cd]ef"`
>输出：`"abcabccdcdcdef"`

示例 4：

>输入：`s = "abc3[cd]xyz"`
>输出：`"abccdcdcdxyz"`

**提示：**

- `1 <= s.length <= 30`
- `s` 由小写英文字母、数字和方括号 `'[]'` 组成
- `s` 保证是一个 **有效** 的输入。
- `s` 中所有整数的取值范围为 `[1, 300]`

# 解题思路与实现

- **num_stack**  
    保存每一层的数字（重复次数）
- **str_stack**  
    保存每一层对应的“之前的字符串”
- **cur_str**  
    当前层正在构造的字符串
- **num**  
    当前解析的数字（可能是多位）

| **维度**    | **分析内容**                                                                                                                                                                                                                          |
| --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **核心思路**  | **栈的状态机模型**。<br>1. **数字处理**：连续读取数字并计算 `multi`，解决如 `100[a]` 的情况。<br>2. **遇到 `[`**：进入嵌套。将当前的“外部字符串”和“重复倍数”分别压栈，并清空当前字符串开始记录括号内的内容。<br>3. **遇到 `]`**：结束嵌套。弹出最近的倍数和外部字符串，进行拼接：`外部字符串 + 倍数 * 括号内内容`。<br>4. **普通字符**：直接累加到当前结果 `res` 中。 |
| **关键点**   | `str_stack.append(res)`：这里的 `res` 存储的是进入当前 `[` 之前的“前缀”内容，这一步是处理嵌套的核心。                                                                                                                                                             |
| **时间复杂度** | $O(S)$<br>其中 $S$ 是解码后生成的字符串长度。每个字符都会被遍历并构建。                                                                                                                                                                                       |
| **空间复杂度** | $O(N)$<br>栈的深度取决于嵌套的层数，最坏情况下与原始字符串长度成正比。                                                                                                                                                                                          |
## 方法一：

```python
class Solution:
    def decodeString(self, s: str) -> str:
        num_stack = []
        str_stack = []
        res = ""
        i = 0
        n = len(s)
        while i < n:
            char = s[i]
            if char.isdigit():
                multi = 0
                while s[i].isdigit():
                    multi = multi * 10 + int(s[i])
                    i += 1
                num_stack.append(multi)
                continue
            elif char == '[':
                str_stack.append(res)
                res = ""
            elif char == ']':
                k = num_stack.pop()
                prev_str = str_stack.pop()
                repeated_str = res * k
                res = prev_str + repeated_str
            else:
                res += char
            i += 1
        return res

```
