🏷中等 #字符串 #动态规划 #回溯 

数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效的** 括号组合。

示例 1：

>输入：`n = 3`
>输出：`["((()))","(()())","(())()","()(())","()()()"]`

示例 2：

>输入：`n = 1`
>输出：`["()"]`

**提示：**

- `1 <= n <= 8`

# 解题思路与实现

- 用 backtracking，不断往字符串里加 '(' 或 ')'，分别减少 left 和 right，但必须保证 right ≥ left 才能加 ')'。当 left 和 right 都为 0 时，得到一个合法方案。

| **维度**    | **分析内容**                                                                                                                                                                                                                        |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **核心思路**  | **受限回溯 (Constrained Backtracking)**。<br>1. **左括号优先**：只要左括号还有剩余 (`left > 0`)，就可以选左括号。<br>2. **有效性约束**：只有当剩余的右括号多于左括号 (`right > left`) 时，才能选右括号（这意味着此时已放置的左括号更多，能匹配掉一个右括号）。<br>3. **终止条件**：左右括号都用光了 (`left == 0 and right == 0`)。 |
| **空间复杂度** | $O(n)$<br>主要开销是递归栈深度，最大深度为 $2n$。                                                                                                                                                                                                |
| **时间复杂度** | $O(\frac{4^n}{\sqrt{n}})$<br>这是 **卡特兰数 (Catalan Number)** 的复杂度。虽然看起来是指数级，但因为有强约束剪枝，实际生成的有效路径远少于 $2^{2n}$。                                                                                                                       |
| **优点**    | 1. **剪枝极佳**：通过 `right > left` 过滤掉了所有非法前缀，避免了生成无效括号后再去判断的开销。<br>2. **状态清晰**：直接利用剩余数量进行递归，逻辑非常硬核。                                                                                                                                 |

## 方法一：

```python
class Solution:
    def generateParenthesis(self, n: int) -> List[str]:
        res = []
        path = []

        def backtrack(left, right):
            if left == 0 and right == 0:
                res.append("".join(path))
                return
            if left > 0:
                path.append('(')
                backtrack(left-1,right)
                path.pop()

            if right > left:
                path.append(')')
                backtrack(left,right-1)
                path.pop()
                
        backtrack(n,n)
        return res
```
