🏷中等 #递归 #链表 #数字

给你两个 **非空** 的链表，表示两个非负的整数。它们每位数字都是按照 **逆序** 的方式存储的，并且每个节点只能存储 **一位** 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

**示例 1：**

![ | 350](https://assets.leetcode-cn.com/aliyun-lc-upload/uploads/2021/01/02/addtwonumber1.jpg)

输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807.
示例 2：

输入：l1 = [0], l2 = [0]
输出：[0]
示例 3：

输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]

**提示：**

- 每个链表中的节点数在范围 `[1, 100]` 内
- `0 <= Node.val <= 9`
- 题目数据保证列表表示的数字不含前导零

# 解题思路与实现

- 同时遍历 `l1` 和 `l2`；
- 对应节点的值相加，再加上“进位 carry”；
- 创建新节点存储当前的个位数；
- 移动指针；
- 循环结束后，如果还有进位，补一个节点。

| **维度**    | **详细分析**                                                                                                                                         |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| **核心思路**  | 同时遍历两个链表，逐位相加。利用 `carry` 变量存储进位，`dummy` 节点简化链表构建。                                                                                                |
| **伪代码实现** | `while l1 or l2 or carry:`<br>1. 取 `l1, l2` 当前值（若为空则为0）<br>2. `sum = v1 + v2 + carry`<br>3. 更新 `carry = sum // 10`, `curr.next = Node(sum % 10)` |
| **时间复杂度** | $O(\max(M, N))$，其中 $M, N$ 分别为两个链表的长度。                                                                                                            |
| **空间复杂度** | $O(\max(M, N))$，返回的新链表长度最多为 $\max(M, N) + 1$。                                                                                                    |

## 方法一：迭代

```python
class Solution:
    def addTwoNumbers(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(0)
        curr = dummy
        carry = 0
        while l1 or l2 or carry:
            v1 = l1.val if l1 else 0
            v2 = l2.val if l2 else 0
            total = v1 + v2 + carry
            carry = total//10
            curr.next = ListNode(total%10)
            curr = curr.next 

            if l1: l1 = l1.next
            if l2: l2 = l2.next

        return dummy.next
```
