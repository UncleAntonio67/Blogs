🏷中等 #数组 #二分查找 #矩阵 

给你一个满足下述两条属性的 `m x n` 整数矩阵：

- 每行中的整数从左到右按非严格递增顺序排列。
- 每行的第一个整数大于前一行的最后一个整数。

给你一个整数 `target` ，如果 `target` 在矩阵中，返回 `true` ；否则，返回 `false` 。

**示例 1：**

![|200](https://assets.leetcode.com/uploads/2020/10/05/mat.jpg)

>输入：`matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 3`
>输出：`true`

**示例 2：**

![|200](https://assets.leetcode-cn.com/aliyun-lc-upload/uploads/2020/11/25/mat2.jpg)

>输入：`matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,60]], target = 13`
>输出：`false`

**提示：**

- `m == matrix.length`
- `n == matrix[i].length`
- `1 <= m, n <= 100`
- `-104 <= matrix[i][j], target <= 104`

# 解题思路与实现

- 二分查找

| **维度**     | **分析内容**                                                                                                                                |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| **核心思路**   | **逻辑一维化 (Virtual Flattening)**。<br>1. **索引映射**：通过数学转换，将一维索引 `mid` 映射回二维坐标 `(row, col)`。<br>2. **二分查找**：在 `0` 到 `m*n - 1` 的范围内进行标准的折半搜索。 |
| **坐标转换公式** | 设矩阵列数为 $n$，一维索引为 $idx$：<br>1. $row = idx // n$ (商为行号)<br>2. $col = idx \% n$ (余数为列号)                                                    |
| **时间复杂度**  | $O(\log(M \times N))$<br>这等价于 $O(\log M + \log N)$，是该问题的理论最优解。                                                                          |
| **空间复杂度**  | $O(1)$<br>只使用了固定的变量，没有开辟额外的存储空间。                                                                                                        |
## 方法一：

```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        m, n = len(matrix), len(matrix[0])
        left, right = 0, m*n -1

        while left <= right:
            mid = (left+right) //2
            row = mid // n
            col = mid % n
            val = matrix[row][col]

            if val == target:
                return True
            elif val < target:
                left = mid + 1
            else:
                right = mid - 1
        return False
```
